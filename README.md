# קורס פיתוח משחקי מחשב
## אוניברסיטת אריאל, המחלקה למדעי המחשב, סמסטר ב ה'תש"ף 

הקורס כולל שני נושאים עיקריים:

* תהליך העיצוב והפיתוח של משחקים ומשחקי-מחשב.
* תיכנות משחקים בשפת סי שארפ ומנוע יוניטי.
